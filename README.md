# 🚀 CampusIQ — AI-Powered Campus Talent Intelligence Platform

A **multi-portal SaaS-style web application** designed to manage and analyze campus talent across students, faculty, and recruiters.

> ⚡ Built using **HTML, CSS, and Vanilla JavaScript**
> ⚡ No backend — powered by **localStorage (browser-based database)**

---

## 🌐 Live Demo

🔗 https://ayushraj102.github.io/CollegeIQ/

---

## 🎯 Key Features

* 🧠 AI-style **Talent Insights & Readiness Scoring**
* 🎓 Student **Portfolio & Skill Profile**
* 👨‍🏫 Teacher Dashboard (**grading + verification + research**)
* 🛡️ Admin Panel (**system control + analytics**)
* 🏢 Recruiter Portal (**talent discovery + leaderboard**)
* 🌙☀️ Light / Dark Mode (modern UX)
* 📊 Dynamic data system using localStorage
* 🎨 Modern UI (glassmorphism + gradients + animations)

## 📁 Project Structure

```
campusiq/
├── index.html        → Login page
├── student.html      → Student portal
├── teacher.html      → Teacher portal
├── admin.html        → Admin dashboard
├── recruiter.html    → Recruiter portal
├── data.js           → localStorage database + logic
├── shared.css        → Global styles (UI system)
└── .nojekyll         → Required for GitHub Pages
```

---

## 🛠️ Tech Stack

* HTML5, CSS3, Vanilla JavaScript (ES6+)
* localStorage (mock database)
* Google Identity Services (optional login)
* CSS Variables (theme system)
* Canvas API (background animation)

## 🚀 Deployment (GitHub Pages)

1. Push project to repository
2. Go to **Settings → Pages**
3. Select:

   * Source → `main`
   * Folder → `/ (root)`
4. Save
